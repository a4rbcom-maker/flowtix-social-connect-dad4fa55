## الهدف
استبدال شاشة `/dashboard/whatsapp/inbox` الحالية بشاشة محادثة بنفس شكل Bot-Xtra: تأخذ كامل الشاشة (بدون شريط الـ DashboardLayout الجانبي والرأسي)، مع زر رجوع للعودة إلى الداشبورد.

## المكوّن المرجعي من Bot-Xtra
ثلاث لوحات قابلة لإعادة الحجم (Resizable):
- **اليمين (RTL):** قائمة المحادثات (`ConversationList`) — بحث، فلاتر (الكل / غير مقروء / AI)، شارات عدد الرسائل، صورة شخصية، مؤشر اتصال أخضر.
- **الوسط:** نافذة الشات (`ChatWindow`) — رأس الجهة (اسم + رقم + حالة الاتصال + أزرار AI/Sound/إغلاق)، خلفية فقاعات WhatsApp اختيارية، فقاعات رسائل صوتية/نصية مع علامات قراءة، شريط أدوات ذكي (تحسين / صياغة / اقتراح ذكي / ردود / جدولة / كتالوج)، Composer متعدد المرفقات.
- **اليسار:** لوحة جهة الاتصال (`ContactPanel`) — أفاتار كبير، اسم، حالة، إجراءات سريعة (حفظ، تلخيص)، تبويبات (معلومات / ملاحظات / وسائط)، تحليل شخصية العميل.

## التغييرات على Flowtix

### 1) Route فل‑سكرين خارج DashboardLayout
- إزالة لف الصفحة بـ `DashboardLayout` في `src/routes/dashboard.whatsapp.inbox.tsx`.
- جعل الجذر `fixed inset-0 z-50 bg-background` ليأخذ كامل الفيوبورت.
- إضافة زر **رجوع** ثابت أعلى اليمين/اليسار (حسب RTL) يستخدم `Link to="/dashboard/whatsapp"` بأيقونة `ArrowLeft/ArrowRight` + نص "خروج من المحادثة".

### 2) إعادة بناء البنية بثلاث لوحات
```text
┌─────────────────────────────────────────────────────────────┐
│ TopBar: [← رجوع]   عنوان "صندوق الوارد"     [🔔][🌙][👤]   │
├──────────────┬──────────────────────────────┬───────────────┤
│  Conv List   │         Chat Window          │  Contact Panel│
│  (بحث+فلاتر) │  (header + messages + comp.) │  (info+tabs)  │
│   25-30%     │           45-50%             │     20-25%    │
└──────────────┴──────────────────────────────┴───────────────┘
```
- استخدام `ResizablePanelGroup` الموجود بالفعل.
- على الموبايل: عرض لوحة واحدة في كل مرة مع تبديل (`list` → `chat` → `contact`) وزر رجوع داخلي بين اللوحات.

### 3) مكوّنات جديدة (نسخ مبسّط مطابق بصرياً لـ Bot-Xtra)
ملفات جديدة تحت `src/components/inbox/`:
- `InboxConversationList.tsx` — قائمة المحادثات بشارات وفلاتر.
- `InboxChatWindow.tsx` — الرأس + سجل الرسائل (فقاعات صوتية، CheckCheck، تاريخ، خلفية WA اختيارية) + Composer.
- `InboxContactPanel.tsx` — بطاقة الجهة + تبويبات معلومات/ملاحظات/وسائط + إجراءات سريعة + تحليل الشخصية (Placeholder).
- `inbox-theme.css` (اختياري) — متغيرات اللون لخلفية WhatsApp.

### 4) ربط البيانات
الاعتماد على نفس الـ server functions الحالية:
`listConversations`, `getConversationMessages`, `sendChatMessage`, `toggleConversationAi`, `markConversationRead` — بدون أي تغيير في الـ backend.

### 5) إدخالات Bot-Xtra البصرية المراد محاكاتها بدقة
- لون primary البنفسجي الموجود في `mem://design/brand` (يطابق Bot-Xtra أصلاً).
- فقاعات الرسائل مع زوايا غير متماثلة، ظل خفيف، توقيت بالعربي.
- شريط إجراءات ذكي بأزرار ملوّنة (تحسين/صياغة/اقتراح ذكي/ردود/جدولة/كتالوج).
- شارة "متصل الآن" خضراء نابضة.

## القيود
- لا تعديل على قاعدة البيانات أو الـ server functions.
- الحفاظ على دعم RTL/LTR وتبديل اللغة.
- اختبار الشاشة بعد التنفيذ على ديسكتوب + موبايل عبر `browser--view_preview`.
