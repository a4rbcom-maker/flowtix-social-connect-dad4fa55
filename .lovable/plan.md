## الهدف
تبسيط صفحة "ربط حساب فيسبوك" بحيث تدعم 3 طرق ربط بشكل واضح وموثوق، مع تعليمات صورية خطوة بخطوة، وحل مشاكل فتح الروابط وفشل الحفظ.

## الطرق الثلاث

| الطريقة | الاستخدام | الصعوبة | الموثوقية |
|---|---|---|---|
| **A. كوكيز (مُوصى به)** | نسخ JSON من إضافة Cookie-Editor | سهل جداً | عالية (أسابيع) |
| **B. توكن Graph API (رسمي)** | Access Token من Meta للنشر في الصفحات | متوسط | عالية (شهرين) |
| **C. إيميل/باسوورد** | تسجيل دخول البوت تلقائياً | الأسهل بصرياً | منخفضة (فيسبوك قد يحظر) |

## التغييرات

### 1. واجهة اختيار الطريقة (`dashboard.facebook.tsx`)
- 3 كروت اختيار بصرية بدلاً من اثنين، كل كرت يوضح:
  - أيقونة + اسم الطريقة + شارة "موصى به / رسمي / تجريبي"
  - جملة واحدة "متى تستخدمها"
  - الصعوبة + المدة قبل انتهاء الصلاحية
- إضافة `connectionMode: "cookies" | "token" | "credentials"` بدل اثنين

### 2. تعليمات واضحة لكل طريقة
- **كوكيز**: شريط خطوات مرقم (1→2→3→4) مع رابط مباشر لتثبيت Cookie-Editor (Chrome + Firefox)، صورة توضيحية للزر "Export → JSON"، ثم لصق في حقل JSON
- **توكن**: شريط خطوات مع نسخ تلقائي للصلاحيات وفتح Graph API Explorer
- **إيميل/باسوورد**: تحذير واضح بأنها تجريبية + حقول الإيميل والباسوورد + 2FA اختياري + توضيح أن فيسبوك قد يطلب تأكيد

### 3. حل مشكلة "فتح الروابط ما بتشتغل"
- استبدال زر "افتح" بمكون موحّد يعرض:
  - رابط `<a target="_blank" rel="noopener">` حقيقي قابل للنقر (مش onClick فقط)
  - زر "نسخ الرابط" دائماً بجانبه كـ fallback
  - عرض الرابط الكامل تحت الزر مباشرة (للنسخ اليدوي)
- إزالة الحجب الذي يسببه preview iframe

### 4. حل مشكلة "الحفظ فشل"
- **كوكيز**: عرض تشخيص فوري قبل الحفظ — يبيّن أي كوكيز مفقودة من القائمة المطلوبة (`c_user, xs, fr, datr, sb`) برسالة عربية واضحة
- **توكن**: قبل الحفظ، فحص أولي للتوكن عبر `/me` ويعرض المالك + الصلاحيات الممنوحة + الناقصة
- **إيميل/باسوورد**: استدعاء `addBotAccount` بـ `method: "credentials"` (موجود بالفعل في الـ backend)
- رسائل خطأ مفصّلة عربياً (بدل "فشل") تشرح السبب والحل

### 5. شريط "حالة الربط الحالي"
عند نجاح أي طريقة، عرض كرت موحّد يبيّن:
- نوع الربط المستخدم
- اسم الحساب
- تاريخ انتهاء الصلاحية (إن وُجد)
- زر اختبار سريع (Dry-run)
- زر فصل / تبديل الطريقة

## الملفات المتأثرة
- `src/routes/dashboard.facebook.tsx` — UI الرئيسي (تعديل واسع)
- `src/lib/fb-bot.functions.ts` — يدعم `credentials` بالفعل، لا تعديل
- `src/lib/facebook.functions.ts` — يدعم التوكن بالفعل، لا تعديل
- مكون مشترك جديد `src/components/dashboard/ExternalLinkRow.tsx` لعرض الروابط مع زر نسخ

## غير مشمول
- لن أضيف OAuth بنقرة واحدة لأن Lovable Cloud لا يدعم Facebook Provider أصلاً (يحتاج Meta App مُوافَق عليه من فيسبوك — أسابيع مراجعة)
- لن أعدّل قاعدة البيانات
